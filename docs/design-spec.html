Navigation Structure

Root Navigator (Stack)
├── Welcome             — unauthenticated landing page
├── Login               — email + password sign in
├── Register            — new account creation + terms
├── ForgotPassword      — password reset by email
├── ProfileSetup        — diet + allergies setup (2 steps)
└── Main (Bottom Tabs)
    ├── Tab 1: Home     — dashboard + scan history
    ├── Tab 2: Scan     — full-screen camera modal
    └── Tab 3: Profile  — settings + account + usage
            └── Result      — SAFE / CAUTION / NOT_SAFE
            └── EditPersonal — name + bio + avatar

Welcome Screen

  • "VeganLand" logo in serif + tagline "Scan, trust, eat well."
  • Subtitle: "Analyze any product with AI..."
  • 3 feature rows: Take a photo / AI analyzes / Instant result
  • CTA: "Get Started" — primary green button, full width
  • Secondary: "I already have an account" — text link
  • Language picker: PT / EN / DE / FR / IT / ES flag buttons
  • Beta ribbon: top-right corner, green "BETA" badge
Login

Login Screen

  • Tagline header: "Your vegan life, simpler"
  • Email input field
  • Password input field + "Forgot password?" link
  • "Sign In" button — primary green, full width
  • Conditional warning banner (amber): email not confirmed → "Resend email" button
  • "Don't have an account? Create account" link
  • Language picker row
Register

Register Screen

  • Email, Password, Confirm Password fields
  • Terms checkbox (required): "I agree to the Terms & Conditions and Privacy Policy"
  • Terms / Privacy links open in in-app browser
  • "Create Account" button — primary green, full width
  • "Already have an account?" link
  • On success: navigates to Check Email screen

  • Step indicator: "Step 2 of 2 · Sensitivities"

HomeAuth required

Home Screen (Tab 1)

  • Header: "VeganLand" serif + "What are we scanning today?" + scan count badge (dark)
  • Profile card (glassmorphism): diet icon circle + name + description + allergy chips + settings button
  • Scan button (dark card, green glow): scan icon circle + "Scan Product" + subtitle + arrow
  • Recent Scans (up to 5): left colored border + status icon + status label + product name + date

ScanFull screen

Scan Screen

  • Camera fills entire screen (back camera)
  • Top bar: ✕ close (left) + "Scan Product" title
  • Frame: 280×280px with amber corner brackets
  • Hint text below frame
  • Rotating tip pill (3.5s interval): barcode / ingredients / product front tips
  • Bottom bar: Gallery button (left) + Capture button (80px circle, center)
  • Analyzing overlay: dark background + white card + AI icon + spinner
  • Error card overlay: dark background + white card + error text + OK dismiss button
Result

Result Screen

  • Header: product name + source badge
  • Status banner: full-width colored strip — SAFE / CAUTION / AVOID
  • Analysis card: AI explanation text
  • Ingredients card: chip grid; flagged ingredients in red; always visible
  • Allergens card (amber, #FFF8ED): allergen chips; always visible; "No allergens detected" if empty
  • Concerns card: problematic ingredients list (if any)
  • No issues card (green): shown if SAFE + no concerns
  • Bottom: "Scan Another" button
